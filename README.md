# Markdown Previewer

This project is markdown editor with a live preview. When you enter GitHub flavored markdown into the editor, the text is rendered as HTML in the previewer as you type.

![screenshot](./Screenshot.png)

## :package: Built With

    - HTML5
    - CSS3
    - React
